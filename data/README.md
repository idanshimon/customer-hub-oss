# Data

This directory holds workspace-level datasets — shared inputs that span across customers.

## Structure

- **`source/`** — Original source files (manually maintained, version controlled)
- **`exports/`** — CSV conversions of source files (auto-generated by scripts)
- **`outputs/`** — Derived reports and analysis results (auto-generated)

## Rules

- Scripts must **never** modify files in `source/` — source files are manually maintained only.
- Scripts read from `exports/` and write to `outputs/`.
- Do not treat `data/` files as customer context; they are shared inputs.
